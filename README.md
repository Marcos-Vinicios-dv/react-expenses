## react-expenses

<p>Trata-se de um app desenvolvido como trabalho prático do bootcamp IGTI, nessa aplicação uma tabela é preenchida com informações de uma fake API e filtradas por ano e mês.</p>

### Imagem da aplicação:

### Tecnologias e bibliotecas utilizadas:

<b>Linguagem</b>
- Typescript(React).

<br>

<b>Bibliotecas</b>
- <a href="https://axios-http.com/docs/intro" >axios</a> (requisições a api);
- <a href="https://github.com/typicode/json-server" >json-server</a> (fake API);
- <a href="https://reactrouter.com/web/guides/quick-start" >react-router-dom</a> (navegação);
- <a href="https://material-ui.com/pt/getting-started/installation/" >material-ui</a> (componentes);


## Inicializar projeto

<p>Para inicializar esse projeto em sua máquina será necessário usar os seguinte comandos(como o node instalado):</p>

- Iniciar Back-end

    Instalar depêndencias 
    ```npm install```

    Iniciar server localmente 
    ```npm start```

- Iniciar Front-end

    Instalar depêndencias 
    ```npm install```

    Inicializar aplicação localmente
    ```npm start```
